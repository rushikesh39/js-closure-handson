Q2:- 

        "let count = 0;
        (function () {
          if (count === 0) {
            let count = 1;
            console.log(count); // What is logged?
          }
          console.log(count); // What is logged?
        })();"

variable count using let keyword and initializes its value to 1. This new variable has block scope, so it only exists within the block of the if statement. Therefore, the console.log statement within the if block logs 1.

The console.log statement outside of the if block logs 0 because it is referencing the count variable defined outside the function (which has not been updated by the let statement).

So, the output will be: 1 and 0.

Q3:- 

        "for (var i = 0; i < 3; i++) {
            setTimeout(function log() {
              console.log(i); // What is logged?
            }, 1000);
          }"

The code will log the number 3 three times to the console. This is because the for loop initializes the variable i to 0 and increments it by 1 each time through the loop until it reaches a value of 3. During each iteration of the loop, a function is defined and passed as an argument to setTimeout(). This means that each setTimeout call will create a closure that captures the current value of i. However, the setTimeout() function schedules the execution of its callback function after a delay of 1000 milliseconds, meaning that the loop will finish executing before the first timer has a chance to print its value. Therefore, when the three timer functions finally execute, the value of i will be 3 in each case, since that is the value of i when the loop exits. So the code will log 3 to the console three times.
